## Como executar a aplicação
* Tenha o Python. <br>
* Clone o repositório. <br>
* Abra a pasta Portfolio <br>
* No terminal:
```powershell
#Crie um ambiente virtual
python -m venv venv

#Ative o ambiente virtual:
./venv/Scripts/activate

#Instale o Flask:
pip install flask

#Execute o programa (na pasta src/)
python app.py
```
